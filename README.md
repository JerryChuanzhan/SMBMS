# SMBMS  超市订货管理系统

**（Supermarket  indent management system）**

基于 Java Servlet 的初级超市订货管理系统   

个人学习Java的历程，逐步从Servlet 到 WebMVC ，到 spring boot  ，然后SpringCloud ，将自己学习的东西逐步完善这个项目 。
